# SSRVPN 用户指南

SSRVPN 是一款基于 Clash.Meta 内核的安全 VPN 客户端，支持 Android / macOS / Windows 三平台。

当前节点与路由策略明确为 IPv4-only，不支持 IPv6 节点、IPv6 强制代理 IP 或 IPv6 出口。

## 快速开始

### 1. 添加订阅

首次启动会自动弹出「添加订阅」窗口。你也可以稍后在「订阅」页面添加。

**方式一：粘贴订阅链接**

在订阅页面点击添加按钮，粘贴你的 HTTPS 订阅链接（通常以 `https://` 开头），点击确定。

**方式二：粘贴 SSR 链接**

部分服务商会提供 `ssr://` 开头的链接，直接粘贴即可。

### 2. 连接节点

1. 返回「主页」，你可以看到所有可用节点
2. 点击节点可查看详情、修改配置、测试延迟
3. 点击底部中央的**大圆形按钮**即可连接
4. 连接成功后按钮变为激活状态，状态栏/托盘图标会变化

### 3. 切换节点

连接状态下，点击任意节点可直接切换，无需断开重连。

### 4. 解锁检测

「解锁」页面可以测试当前节点是否支持以下流媒体服务：

- Netflix — 测试 Originals / 地区内容
- Disney+ — 测试可用区域
- YouTube Premium — 测试地区支持
- ChatGPT Web / iOS — 测试 OpenAI 限制
- Gemini / Claude — 测试 AI 服务可用性
- Prime Video / Spotify / TikTok — 测试流媒体

点击「开始检测」按钮，约 10-30 秒出结果。✅ 绿色表示解锁，❌ 红色表示被屏蔽。

## 各平台特性

### Android

- **快速设置磁贴**：下拉通知栏可看到 SSRVPN 开关，一键连接/断开
- **后台运行**：VPN 连接后在通知栏常驻，不会被系统杀死
- **TUN 模式**：默认开启，所有应用流量自动走代理

### macOS

- **菜单栏图标**：随时查看连接状态，右键菜单快速操作
- **最小化到托盘**：关闭窗口后自动隐藏到菜单栏
- **系统代理**：连接时自动设置 macOS 系统代理，断开自动清除
- **窗口尺寸**：支持自由缩放，最小 860×540

### Windows

- **系统托盘**：右下角图标右键控制连接
- **绿色免安装**：解压即用，无需安装
- **便携模式**：配置文件和数据保存在程序目录

## 设置说明

| 设置项 | 说明 |
|--------|------|
| 代理端口 | HTTP/SOCKS5 代理端口（默认 7890） |
| API 端口 | Clash API 控制端口（默认 9090） |
| 延迟超时 | 节点延迟测试超时时间（默认 5 秒） |
| 代理模式 | Rule（规则）/ Global（全局）/ Direct（直连） |
| 强制代理网站 | 自定义需要走代理的域名/IP |
| 暗色模式 | 切换深色/浅色主题 |

## 常见问题

**Q: 连接后无法上网？**
A: 检查节点是否在线。在主页点击节点测试延迟，延迟正常则节点可用。如果所有节点都超时，请刷新订阅。

**Q: 解锁检测失败？**
A: 解锁检测需要当前节点已完成连接。如果检测全部失败，可能是代理端口未正确设置，请检查「设置 → 代理端口」。

**Q: macOS 关闭窗口后找不到应用？**
A: 应用默认最小化到菜单栏（右上角）。查看菜单栏的 SSRVPN 图标，右键点击可选「显示窗口」或「退出」。

**Q: 如何更新订阅？**
A: 进入「订阅」页面，下拉刷新或点击刷新按钮。建议每 24 小时刷新一次以确保节点最新。

**Q: Android 上部分应用不走代理？**
A: Android TUN 模式默认排除局域网地址（192.168.x.x / 10.x.x.x）。如需局域网内服务走代理，请关闭 TUN 模式改用代理模式。
